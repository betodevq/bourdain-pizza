# bourdain_pizza

## Prerequisites

- npm
- node
- docker
- docker-compose

### How to run API

1 - `cd server/src`

2 - `npm i`

3 - `docker-compose up -d`

4 - `npm run migration:run`

5 - `npm run dev`

And the server wil be running on port 3000


### How to run APP

1 - `cd client/src`

2 - `npm i` 

3 - `npm run start`
